
Todo List App (Список задач)

Простое и удобное приложение для управления задачами, созданное на React. Позволяет добавлять, отслеживать и управлять повседневными делами.

Возможности

- ✅ Добавление новых задач
- ❌ Удаление задач
- 🔄 Переключение статуса выполнения задачи
- 📊 Счётчик активных задач в реальном времени

Технологии

- React
- Хук useState
- CSS

Быстрый старт

1. Клонируйте репозиторий
2. Выполните `npm install`
3. Выполните `npm start`

Как это работает

Введите задачу в текстовое поле, нажмите кнопку «Сохранить» чтобы добавить её. Нажмите на задачу, чтобы отметить её как выполненную, или используйте кнопку удаления. Счётчик задач обновляется автоматически.
